# Storyboard Studio 수정 요청 사항

## 프로젝트 개요
그림 작가와 원활한 커뮤니케이션을 위한 스토리보드 협업 웹사이트의 기능 개선

## 수정 요청사항

### 1. 초안/아트워크 선택 방식 변경
- 현재 문제: 두 개 모두 선택 시 아무것도 표시되지 않음
- 해결방안: 라디오 버튼 방식으로 변경하여 한 번에 하나만 선택 가능하게 수정

### 2. 비교모드 UI 개선  
- 현재 문제: 버튼을 클릭해야만 이미지가 표시됨
- 해결방안: 비교모드 활성화 시 초안과 아트워크를 양옆에 동시 표시

### 3. 씬 추가 기능 구현
- 새로운 씬을 추가할 수 있는 버튼 추가
- 씬 추가 모달 창 구현
- 씬 목록 관리 기능

### 4. 주석 그리기 기능
- 초안/아트워크 위에 그리기 가능한 오버레이 캔버스 추가
- 원본 이미지는 수정하지 않고 주석 레이어로 관리
- 댓글과 연동되어 주석 표시/숨김 가능

### 5. 워크플로우 개선
- 수정 요청 후 "수정 완료" 버튼 추가
- 상태 전환 플로우 명확화
- 진행 상황 추적 가능한 UI

## 기술 스택
- Next.js 14.2.2
- React 18
- TypeScript
- Tailwind CSS

## 우선순위
1. 초안/아트워크 선택 및 비교모드 수정 (긴급)
2. 씬 추가 기능 (높음)
3. 주석 그리기 기능 (중간)
4. 워크플로우 개선 (중간)
